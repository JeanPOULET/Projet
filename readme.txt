***************
* Commandes à faire à la racine du projet
*  --Projet 
*       |--public
*       |--index.js
*       
npm init
npm install express --save
npm install socket.io --save
*
**************

Une fois les commandes réaliser vous pouvez lancer le serveur à la racine du projet avec la commande : node .

Dès que le serveur est lancé, vous pouvez dans votre navigateur favori ouvrir un ou plusieurs onglet(s) la page localhost:8080 .
