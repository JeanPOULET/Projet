***************
* Commandes à faire à la racine du projet dans cet ordre
npm init
npm install express --save
npm install socket.io --save
*
**************

Une fois les commandes réalisées vous pouvez lancer le serveur depuis un terminal à la racine du projet avec la commande : node .

Dès que le serveur est lancé, vous pouvez dans votre navigateur favori ouvrir un ou plusieurs onglet(s) la page localhost:8080 ou adresse_ip_hebergeur_serveur:8080 depuis un autre ordinateur entant sur le même réseau local.
